# DraftMate —— 给你的一页说明

## 它是什么

读你屏幕上的聊天，帮你想下一句怎么回；草稿你过目后自己复制粘贴，AI 不替你发任何消息。

## 隐私承诺

- **截图不出你的电脑。** 默认用本地模型（Ollama），识别和生成全在本机完成，不联网上传任何内容。
- **永不替你发送消息。** 它只出草稿，发不发、改不改，每一句都是你自己决定、自己粘贴。
- 左上角的「监控」开关默认关闭；打开后它会盯着你**当前打开的那个对话**，每隔几秒截屏一次、对方在这个会话里发新消息就自动出草稿——截图同样只在你本机处理。注意它只能看到你点开的会话：别的联系人来消息，微信自己会提醒你，点开后监控自动接手。
- **导入历史**（设置 ⚙ 里）会**自动滚动**当前对话来读完历史——这是唯一会动鼠标的地方，纯只读浏览、由你点按钮触发，绝不输入文字、绝不点发送。要用它得在 系统设置 → 隐私与安全性 → **辅助功能** 里也勾上 DraftMate（不用导入历史就不需要这个权限）。
- **没有遥测。** 界面右上角的「已读取 N 次」只存在你本机；周报全靠你自愿截图发群里，不发也没关系。

## 安装四步

1. 装 [Ollama](https://ollama.com/download)（本地模型运行器，免费），装完打开终端拉模型：
   - `ollama pull qwen2.5:7b` —— 生成回复用，约 4.7GB，**必装**；
   - `ollama pull qwen2.5vl:7b` —— 截图识别的兜底，约 6GB，可选（磁盘紧张可跳过）。
2. 把 `DraftMate.app` 拖进「应用程序」。首次打开用 **右键 → 打开**（App 未签名，直接双击会被 macOS 拦下）。
3. 系统设置 → 隐私与安全性 → **屏幕录制** → 勾上 DraftMate，然后**重新打开**它。
4. 打开桌面微信、停在想回的聊天窗口，回到 DraftMate 点「**读取**」（或按 ⌘R）——左边出现截图、右边出现 2–3 条候选回复，就成功了。

> 默认读的是桌面微信。想用在别的聊天 App：改 `~/Library/Application Support/DraftMate/config.yaml` 里的 `app_name`（遇到问题群里喊我）。

## 已知限制

- 仅 macOS；目标聊天窗口别最小化。
- OCR 对复杂排版、深色主题、花哨表情可能识别不全——读错了把窗口拉大一点再读一次。
- 本地小模型的建议质量"够用不惊艳"；语气不对就点 ↻ 换个说法，或在右上角设置里把"对方信息 / 目标"填了，效果差很多。

## 反馈方式

- 微信群里直接说：用着别扭的地方、离谱的识别/建议（截图发来）、想要的功能。
- 每周日我会在群里问一句「本周谁用过」，自愿把右上角「已读取 N 次」角标截图发我就行。
